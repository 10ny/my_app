# コーヒーメモアプリ

## 制作背景・目的


自分が自宅でコーヒを飲むことが多く、今までに飲んだコーヒ豆をわかりやすく記録したいと思ったこと。
さらに他の人がどんな豆を飲んでいるのか知りたい、同じ思いの人いるはずだと思い、作成に至りました。


## アプリケーション概要


_コーヒー豆メモアプリにあるような機能を投稿として記録できる_ アプリケーション。
