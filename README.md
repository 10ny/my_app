## オリジナルアプリ作成の経緯

家でコーヒを飲むことが多く、好みのコーヒ豆を見やすくわかりやすく記録したいと思ったこと。
さらに他の人がどんな豆を飲んでいるのか知りたい、同じ思いの人いるはずだと思い、作成に至りました。
